# Foundation Standards

These are the baseline standards that apply to all work in this workspace. Specific steering files handle the details — this file sets the mindset.

## Problem Solving

Follow #[[file:.agent/skills/personal/problem-solving/SKILL.md]]

- Break complex problems into smaller tasks before jumping to code
- Define the problem clearly before proposing solutions
- Avoid assumptions — verify first

## Research

Follow #[[file:.agent/skills/personal/research/SKILL.md]]

- When investigating unfamiliar topics, extract key insights and actionable takeaways
- Summarize findings clearly, remove fluff

## General Principles

- Read existing code before writing new code — match the project's style
- Prefer simple solutions over clever ones
- Don't overengineer — deliver the MVP first, iterate later
- Explain tradeoffs when making recommendations
- Flag risks early, don't bury them
- When in doubt, ask — don't guess
