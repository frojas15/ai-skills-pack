# Problem Solving Skill

Goal:
Solve complex problems step-by-step.

Steps:
1. Define problem clearly
2. Break into smaller tasks
3. Solve each part
4. Combine solution

Rules:
- Keep it logical
- Avoid assumptions

Output:
- Step-by-step solution
